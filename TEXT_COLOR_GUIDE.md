### **Method 1: Using Color Names**

### **Method 2: Using Hex Codes**
Use 6-digit hex color codes:
- `#ff0000` (red)
- `#00ff00` (green) 
- `#0000ff` (blue)
- `#ff6600` (orange)
- `#9900cc` (purple)

Use CSS rgb() format:
- `rgb(255, 0, 0)` (red)
- `rgb(0, 255, 0)` (green)
- `rgb(0, 0, 255)` (blue)
